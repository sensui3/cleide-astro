# 🛠️ Guia de Skills: Cleide Sarkis (Astro)

Este arquivo foi criado para facilitar o acesso e a utilização das ferramentas (skills) que mais fazem sentido para este projeto. Com base na análise da stack tecnológica (Astro 5, UnoCSS, SEO), selecionei as 10 skills primordiais.

---

## 🚀 Como usar estas Skills?
Você pode me pedir para aplicar qualquer uma delas diretamente. Por exemplo:
- *"Faça uma auditoria usando a skill astro-optimizer-auditor"*
- *"Melhore o SEO deste componente com a skill seo-content-writer"*
- *"Revise a UI seguindo os padrões da skill ui-ux-pro-max"*

---

## 🔝 Top 10 Skills para este Projeto

| Skill | Finalidade | Por que usar? |
| :--- | :--- | :--- |
| **`astro-optimizer-auditor`** | Auditoria e Performance Astro | Este projeto usa **Astro 5**. Esta skill é a mais completa para garantir que o site esteja rápido e siga as melhores práticas da versão atual. |
| **`seo-audit`** | Diagnóstico SEO | O projeto menciona explicitamente o foco em SEO (`astro-seo`, `sitemap`). Use para encontrar falhas de indexação e autoridade. |
| **`ui-ux-pro-max`** | Design e Experiência do Usuário | Garante que o design seja premium, acessível e sem "gaps" visuais. Útil para refinar os componentes em `src/components`. |
| **`web-performance-dominator`** | Velocidade Extrema | Além do Astro, esta skill foca em métricas específicas como LCP e CLS, essenciais para uma boa experiência e ranking no Google. |
| **`unocss-expert`** | Estilização Eficiente | (Referenciando padrões Tailwind/UnoCSS). Como você usa **UnoCSS**, esta skill ajuda a manter o código CSS limpo e performático. |
| **`typescript-expert`** | Qualidade de Código | O projeto usa TypeScript (`uno.config.ts`). Use para evitar erros de tipagem e melhorar a manutenção. |
| **`programmatic-seo`** | Escala de Páginas | Se o site da Cleide crescer com muitos artigos ou termos de psicanálise, esta skill ajuda a criar páginas em massa de forma inteligente. |
| **`schema-markup`** | Dados Estruturados | Essencial para que o Google exiba "Rich Snippets" (estrelinhas, perguntas frequentes) nos resultados de busca da Cleide. |
| **`content-creator`** | Redação Especializada | Útil para criar textos que convertam e sejam educativos sobre psicanálise e psicologia. |
| **`clean-code`** | Arquitetura | Mantém a pasta `src/` organizada e fácil de navegar à medida que o projeto cresce. |

## 💡 Dica de Fluxo (Workflow)
Sempre que for iniciar uma nova funcionalidade ou página:
1. Comece com **`ui-ux-pro-max`** para planejar o design.
2. Implemente usando **`typescript-expert`**.
3. Valide o resultado final com **`astro-optimizer-auditor`** e **`seo-audit`**.
